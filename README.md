# campus-network-design
A uni campus network design and implementation project on Cisco Packet Tracer.

Technology Implemented:
<br>
• Hierarchical network model (3 tiers)
<br>
• IP subnetting
<br>
• VLAN
<br>
• Router on a stick (inter-vlan routing)
<br>
• RIP v2
<br>
• Router-based DHCP
<br>
• DNS server
<br>
• Web server
<br>
• Email server
<br>
• SSH
<br>
• SPT Portfast
<br>
